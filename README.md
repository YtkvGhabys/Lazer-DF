# 📱Protótipo UI/UX — Solução Mobile para Descoberta de Lazer no Distrito Federal

## Sobre o Projeto

Este repositório reúne o protótipo desenvolvido no **Figma** para uma **aplicação mobile** voltada à descoberta de atividades de lazer no Distrito Federal. O projeto nasceu da necessidade de facilitar o acesso da população a experiências culturais, gastronômicas, esportivas e de entretenimento, oferecendo recomendações personalizadas e informações relevantes para apoiar a tomada de decisão dos usuários.

O foco do protótipo é demonstrar a experiência do usuário (UX) e a interface (UI) de uma solução mobile, explorando fluxos de navegação intuitivos, organização eficiente das informações e funcionalidades adaptadas ao uso em smartphones.

---

## Proposta

A solução consiste em um **aplicativo mobile** que reúne, em um único ambiente, diferentes opções de lazer disponíveis em todo o Distrito Federal.

Seu principal diferencial é oferecer recomendações personalizadas de acordo com o perfil e as preferências do usuário, permitindo descobrir tanto estabelecimentos permanentes quanto experiências temporárias por meio de uma interface simples, acessível e intuitiva.

---

## Funcionalidades Planejadas

O protótipo contempla funcionalidades como:

- Tela inicial com recomendações personalizadas;
- Busca integrada com Google Maps;
- Filtros por categoria, localização e preferências;
- Informações detalhadas sobre cada atividade;
- Galeria de imagens;
- Faixa de preço;
- Avaliações e comentários de usuários;
- Selos de diferenciais (acessibilidade, ambiente familiar, cardápio, entre outros);
- Sistema de favoritos;
- Cadastro para salvar preferências e compartilhar feedbacks.
